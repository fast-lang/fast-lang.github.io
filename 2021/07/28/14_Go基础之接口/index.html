14_Go基础之接口

2021-07-28
Go
  • Go基础
有 N 人看过

简介: 接口（interface）定义了一个对象的行为准则，只定义规范，不实现，由具体的对象来实现规范的细节。

接口类型

在 Go 中，接口是一种类型，是一种抽象的类型。

interface 是一组 method 的集合，是鸭子类型编程的一种体现。接口做的事情就像是定义一个协议规则，只要一台机器有洗衣服和甩干的功能，我就称它为洗衣机。不关心数据，只关系方法。

为什么要使用接口

type Cat struct{}

func (c Cat) Say() string {return "喵喵喵"}

type Dog struct{}

func (d Dog) Say() string {return "汪汪汪"}

func main() {
    c := Cat{}
    c.say()
    d := Dog{}
    d.say()
}

这段代码中定义了猫和狗，都会叫，会发现有重复的代码，如果继续加上猪、青蛙等，这段代码会无限重复下去，可不可以当成动物呢？
