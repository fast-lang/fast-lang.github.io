11_Go基础之类型别名和自定义类型

2021-07-20
Go
  • Go基础
有 N 人看过

简介: Go 中和其他语言一样，除了一些基础的数据类型，我们同样可以定义属于我们自己的自定义类型。

Go 中使用的是 type 关键字来定义自定义类型。自定义类型是一个全新的类型，我们可以基于内置的基本类型定义，也可以通过 struct 定义。

例如：

type MyInt int

此时就定义好了一个新的类型: MyInt ，具有 int 的特性。

类型别名

类型别名是 Go 1.9 版本才有的功能。

type TypeAlias = Type

如上，TypeAlias 是 Type 的别名，本质上是没有区别的，如同一个人有中文名、英文名等等，都是指代这个人的。

  • rune 和 byte 就是类型别名，定义如下:
type byte = uint8
type rune = int32

类型定义和类型别名的区别

// 类型定义
type NewInt int

// 类型别名
type MyInt = int

func main() {
    var a NewInt
    var b MyInt

    fmt.Printf("type of a: %T\n", a)   // NewInt 类型
    fmt.Printf("type of b: %T\n", b)   // int 类型

结果可以看出，a 的类型是 main.NewInt , 表示 main 包下定义的 NewInt 类型。b的类型就是 int，所以类型别名智慧出现在代码中，编译完成的时候不会存在 MyInt 类型。
