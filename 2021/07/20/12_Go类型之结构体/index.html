12_Go基础之结构体

2021-07-20
Go
  • Go基础
有 N 人看过

简介: Go 中的基础数据类型可以表示一些事物的基本属性，但是当想要表达一类事物的全部属性时，再用单一的基本数据类型明显就不能满足了，这时候就出现了结构体，可以封装多个基本数据类型，英文名称叫 struct ，可以通过 struct 关键字定义属于我们自己的类型。

结构体的定义

使用 type 和 struct 关键字来定义结构体：


type 类型名 struct {
    字段名   字段类型
    字段名   字段类型
    ...

}
  • 类型名: 用于标识自定义结构体的名称，同一个包内不能重复。
  • 字段名: 表示结构体字段名，结构体中的字段名必须唯一。
  • 字段类型: 表示结构体字段的具体类型。

如定义一个人类:


type person struct {
    name string
    city string
    age int8
}

// 同一类型的字段可以写一行
type person1 struct {
    name, city string
    age int8
}

所以 Go 中的面向对象编程就是由 结构体 实现。

结构体实例化

只有当结构体实例化时，才会真正的分配内存，也就是必须实例化后才能使用结构体的字段。结构体本身也是一种类型，所以我们可以像声明内置类型一样使用 var 关键字声明结构体。

// var 结构体实例 结构体类型

type person struct {
    name string
    city string
    age int8
}

func main() {
    var p1 person   // 实例化
    p1.name = "jack"
    p1.city = "北京"
    p1.age = 28
    fmt.Printf("p1=%v\n", p1)   // p1={jack 北京 28}
    fmt.Printf("p1=%#v\n", p1)  // p1=main.person{name:"jack", city:"北京", age:28}
}

通过例子可以看出：我们是通过 . 来访问结构体的字段(也可以叫做成员变量)，例如 p1.name 和 p1.age 等。

匿名结构体

在定义一些临时数据结构等场景下，还可以使用匿名结构体，即没有命名的结构体，可以用于一次声明。

func main() {
    var user struct{Name string; Age int}  // 此时就是一个匿名结构体
    user.Name = "barry"
    user.Age = 19
    fmt.Printf("%#v\n", user)
}

创建指针类型结构体

指针类型结构体需要使用 new 关键字进行实例化，得到的是结构体实例的地址。

var p2 = new(person)
fmt.Printf("%#T\n", p2)  // *main.person
fmt.Printf("p2=%#v\n", p2) // p2=&main.person{name:"", city:"", age:0}

// 同样可以使用 . 来访问结构体成员
p2.name = "Allen"
p2.age = 29
p2.city = "shanghai"
fmt.Printf("p2=%#v\n", p2)  // p2=&main.person{name:"Allen", city:"Shanghai", age:29}

从打印的结果可以清晰看到 p2 是一个指针。

取结构体的地址实例化

使用取地址操作符 & 对结构体进行取地址操作就相当于对该结构体类型进行了一次 new 实例化操作。


p3 := &person{}
fmt.Printf("%T\n", p3)  // *main.person
fmt.Printf("p3=%#v\n", p3)  // p3=&main.person{name: "", city: "", age:0}

p3.name = "Cool"
p3.age = 40
p3.city = "成都"
fmt.Printf("p3=%#v\n", p3)  // p3=&main.person{name:"Cool", city:"成都", age:40}

这里使用 . 运算，其实相当于：p3.name = "Cool" 等价于 (*p3).name = "Cool", 是一个语法糖。

结构体的初始化

  • 键值对初始化
func main() {
    p4 := person {
        name: "小王子",
        age: 18,
    }
    
    // 有些可以不写，也可以使用 &   最后一行逗号必须有
    p5 := &person {
        name: "小王子",
    }

}
  • 值的列表进行初始化
func main () {
    p6 := person {
            "小王子",
            "上海",
            40,
        }
}

结构体内存布局

结构体实例是占用一块连续的内存。

构造函数

Go 结构体中没有构造函数，我们可以自己实现。如下，实现了 person 的构造函数，因为 struct 是值类型，如果结构体比较复杂的话，值拷贝的性能开销会很大，所以，该构造函数返回的是结构体指针类型。

func newPerson(name, city string, age int8) *person {
    return &person{
        name: name,
        city: city,
        age: age,
    }

}

调用该构造函数:

p9 := newPerson("张三", "北京", 90)
fmt.Printf("p9=%#v\n", p9)  // p9=main.person{name:"张三", city:"北京", age:90}

方法和接收者

Go 中的方法Method是一种作用于特定类型变量的函数，这种特定类型变量称为接收者(Receiver),接收者概念就类似于其他语言的 this 和 self。

方法非常类似于函数的定义，只是在函数名(这里是方法名)的前面增加 (接收者变量 接收者类型)，具体如下:


func (接收者变量 接收者类型) 方法名 (参数列表) (返回参数) {
    函数体
}
  • 接收者变量: 接收者中的参数变量名在命名时，官方建议使用接收者类型名的第一个小写字母，如 Person类型的接收者变量应该命名为 p，Connector 类型的接收者变量应该命名为 c 等。
  • 接收者类型：接收者类型和参数类似，可以是指针类型和非指针类型。
  • 方法名、参数列表、返回参数格式与函数定义相同。
package main
import "fmt"

// 方法定义示例

// 是一个结构体
type Person struct {
    name string
    age int8
}


// 构造函数
func NewPerson (name string, age int8) *Person {
    return &Person{
        name: name,
        age: age,
    }
}

// 为Person定义方法
func (p Person) Dream() {
    fmt.Println("正在做梦...")

}


func main(){
    p1 = NewPerson("小张", 19)
    p1.Drean()

}

指针接收者


func (p *Person) setAge(newAge int8) {
    p.age = newAge
}

值接收者

上面 Dream 的就是值接收者。

值接收者传入到方法中的示例，是值拷贝，所以不能在方法中修改示例属性。

什么时候应该使用指针类型

  • 需要修改接收者中的值
  • 接收者是拷贝代价比较大的大对象
  • 保证一致性，如果某个方法使用了指针接收者，那么其他的方法也应该使用指针接收者

不能给内置类型定义方法，只能给自己包里的构造方法。

结构体的匿名字段

结构体允许其成员字段在声明时没有字段名而只有类型，这种没有名字的字段就称为匿名字段。


// Person 结构体 有两个匿名字段
type Person struct {
    string
    int
}


func main() {
    p1 := Person {
        "小明",
        18,
        }

    fmt.Printf("%#v\n", p1)   // main.Person{string:"小明", int:18}
    fmt.Println(p1.string, p1.int)
}

注意：这里的匿名字段的说法并不代表没有字段名，而是默认会采用类型名作为字段名，又因为结构体要求字段名必须唯一，所以一个结构体中相同类型的匿名字段只能有一个。

嵌套结构体

一个结构体中可以嵌套包含另一个结构体或结构体指针，如下：

// Address 地址结构体
type Address struct {
    Province string
    City     string
}

// User 用户结构体
type User struct {
    Name string
    Gender string
    Address Address
}

func main() {
user1 := User {
    Name: "小王子",
    Gender: "男",
    Address: Address {
            Province: "山东",
             City: "威海",
       },
}

同样的，这里的Address 也可以采用匿名字段的形式。

结构体的”继承”

Go 中使用结构体也可以实现其他编程语言中面向对象的继承。

type Animal struct {
    name string
}

func(a *Animal) mvoe () {
    fmt.Printf("%s会动！\n", a.name)
}

type Dog struct {
    Feet int8
    *Animal
}

func (d *dog) wang() {
    fmt.Printf("%s会汪汪汪"a, d.name)
}


func main() {
    d1 := &Dog{
        Feet: 4,
        Animal: &Animal{
            name: "乐乐",
        },
    }

    d1.wang()
    d1.move()

}

结构体字段的可见性

在结构体中，小写表示私有，大写表示可以公开访问。

结构体与 json 序列化

go 中使用 json 内置包进行转换。

// Student 学生
type Student struct {
    ID int
    Gender string
    Name string
}

// Class 班级
type Class struct {
    Title string
    Students []*Student
}

func main() {
    c := &Class{
        Title: "101",
        Students: make([]*Student, 0, 200),
    }

    for i:=0; i<10; i++ {
        stu := &Student{
            Name: fmt.Sprintf("stu%02d", i),
            Gender: "男",
            ID: i,
         }
        c.Students = append(c.Students, stu)
       
    }

    // json 序列化
    data, err := json.Marshal(c)
    fmt.Println(data)


    // json 反序列化
    str := `{"Title":"101","Students":[{"ID":0,"Gender":"男","Name":"stu00"},{"ID":1,"Gender":"男","Name":"stu01"},{"ID":2,"Gender":"男","Name":"stu02"},{"ID":3,"Gender":"男","Name":"stu03"},{"ID":4,"Gender":"男","Name":"stu04"},{"ID":5,"Gender":"男","Name":"stu05"},{"ID":6,"Gender":"男","Name":"stu06"},{"ID":7,"Gender":"男","Name":"stu07"},{"ID":8,"Gender":"男","Name":"stu08"},{"ID":9,"Gender":"男","Name":"stu09"}]}`
    c1 := &Class{}
    err = json.Unmarshal([]byte(str), c1)
    if err != nil {
        fmt.Println("json unmarshal failed!")
        return
    }
    fmt.Printf("%#v\n", c1)
}

结构体标签

Tag 是结构体元信息，可以在运行时候通过反射的机制读取出来。

`key1:"value1" key2:"value2"`

]
结构体tag由一个或多个键值对组成，键与值使用冒号分割，值使用双引号括起来。同一个结构体字段可以设置多个键值对tag。

type Student struct {
    ID int `json:"id"`
}
