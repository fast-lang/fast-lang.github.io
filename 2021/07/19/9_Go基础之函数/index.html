9_Go基础之函数

2021-07-19
Go
  • Go基础
有 N 人看过

简介: 函数是组织好的、可以重复使用的、用于执行指定任务的代码块。Go 中支持普通函数、匿名函数，能够使用闭包技术，同样，函数在 Go 中也是一等公民。

函数定义

Go 中定义函数使用 func 关键字。 通用格式如下: 

func 函数名(参数) (返回值) {
    
    函数体逻辑

}
  • 函数名: 和变量命名规则相同；
  • 参数: 参数由参数变量和参数变量类型组成，如果有多个参数使用逗号分隔；
  • 返回值: 返回值由返回值变量和返回值类型组成，多个返回值必须使用小括号包起来并使用逗号隔开。
  • 函数体逻辑: 用来实现函数功能的逻辑代码。

函数的参数和返回值可选。比如定义一个没有入参，也没返回值的函数，此时入参和返回值就可以为空。

函数定义示例

定义一个求和函数:

func intSum(x int, y int) int {
    return x + y
}

可变参数

在函数参数类型前面加入…为可变类型参数，接收后在函数内部为切片类型, 可变参数要放在后面。

Go 中没有默认参数。

func intSum(a ...int) {
    fmt.Println(a)  // 切片类型
    ret := 0
    for _, v := range a {
        ret = ret + v
    }
    return ret
}

func intSum(a, b int) (ret int) {
    
}

多返回值

也支持类型简写。

func intSum(a, b int) (x, y int) {
    x := a + b
    y := a - b
    return

}

defer 语句

Go 语言中的 defer 语句会将其后跟随的语句进行延时处理。在 defer 归属的函数即将返回时，将延迟处理的语句按照 defer 定义的顺序的逆序进行执行，最先 defer 的最后执行，最后defer的最先执行，类似于压栈处理。

package main

import "fmt"

// defer 延迟执行
func main() {
    fmt.Println("start ")
    defer fmt.Println(1)
    defer fmt.Println(2)
    defer fmt.Println(3)
    fmt.Println("end ")

}

// start end 3 2  1

由于 defer 的特性，一般用于资源的释放等等一些收尾操作。

作用域

函数中：1. 先查找函数作用域； 2. 再往上层找；类似 python 中的查找方式。也有块变量，类似JavaScript。

var num = 10

func testGlobal() {
    fmt.Println(num)  // 可以在函数作用于访问到全局变量

高级函数

接收、或返回函数的函数称为高级函数。


func add(x, y int) int {
    return x + y
}


func calc(x, y int, op func(int, int), int) int {
    return op(x, y)
}


func main() {

    r1 := calc(100, 200, add)

)

其中op接收的是函数。
